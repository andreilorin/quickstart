# Quickstart Maven Archetype

This archetype generates projects with dependencies that allow for rapid prototyping 

##### Included dependencies:
- [agrona](https://github.com/real-logic/agrona)
- [assertj](https://assertj.github.io/doc/)
- [HikariCP](https://github.com/brettwooldridge/HikariCP)
- [jackson](https://github.com/FasterXML/jackson)
- [java-diff-utils](https://github.com/java-diff-utils/java-diff-utils)
- [java-faker](https://dius.github.io/java-faker/)
- [junit5](https://junit.org/junit5/)
- [lombok](https://projectlombok.org/)
- [mockito](https://site.mockito.org/)
- [owner](http://owner.aeonbits.org/docs/usage/)
- [password4j](https://password4j.com/)
- [slf4j](http://www.slf4j.org/)
- [testcontainers](https://www.testcontainers.org/)
